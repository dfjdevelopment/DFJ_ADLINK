ADLINK Project

Our application allows users to securely connect to ADLINK's database in order to manage profiles.
In order to do this, we used the Symfony which allows us to easily add things like security, validation, authentication and authorisation in one bundle to make the website run more efficiently.
We also made our website a PWA(Progressive web application) which allows for full responsiveness and browser compatibility independent of what device the user is using, allows the use of notifications to alert users, works offline and is safe to use.
