ADLINK Project

Our application allows users to securely connect to ADLINK's database in order to manage profiles.
Our project is written in PHP Framework Symfony 5.2. For communication with the MySQL database we are using ORM packaged which uses Doctrine as an abstraction of PDO.
For restriction of access, authorisation and authentication we are using built- in features of Symfony security core which uses user roles for restriction of access. Those roles could be changed, questioned at any point of the development process.
For handling routes, we are using Symfony’s routing functionality which we separate for every route. These routes have functions that pass the main information for the logic to its service parts where the rest of the logic is done. At the end we render the application with a message if the previous part happened successfully or not through build in flash messages.
For handling form’s render, validation and submission we are using Form builder type classes which were designed for all things above. We declare how form would be created in inputs and fields. Then we use a function to create a view in html and pass it as a parameter through the routing package. This way we can use advanced forms of security like CSRF token. For rendering the view part of the website, we are using template engine twig. This powerful building blog is allowing us to break down the parts of the website for clear usage and no repetitive code. Twig can also allow us to print variables, forms, arrays of any kinds with filters and more.
Finally, we use bootstrap and bootstrap-tables for styling and for browser functionality JavaScript and JQuery.
